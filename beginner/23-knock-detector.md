# 23 - Knock Detector

Detect physical taps or knocks using a Piezo Vibration Sensor and light up an LED.

## Goal
Learn how to read a piezo sensor as an analog input to detect mechanical vibrations and trigger a digital output when a set threshold limit is exceeded.

## What You Will Build
Tapping the piezo sensor on the canvas generates a voltage spike. If the tap is strong enough to exceed the threshold limit, the LED on pin D13 flashes ON for 100 ms.

**Why A0 and D13?** Pin A0 measures the fast transient voltage spikes from the piezo element. Pin D13 drives the visual notification LED.

## Wiring
| Component | Component Pin | Arduino Pin | Notes |
| --- | --- | --- | --- |
| Piezo Sensor | POS (Positive) | A0 | Analog signal connection |
| Piezo Sensor | NEG (Negative) | GND | Ground reference |
| LED | A (Anode, longer leg) | D13 | Output signal connection |
| LED | C (Cathode, shorter leg) | GND | Ground reference |

> On physical hardware, you should place a **1 Megaohm resistor** in parallel with the piezo sensor (connecting POS directly to NEG) to absorb excess voltage spikes.

## Code
```cpp
const int PIEZO_PIN = A0;
const int LED_PIN   = 13;

// Threshold for what we consider a "knock" (0 to 1023)
const int KNOCK_THRESHOLD = 100;

void setup() {
  pinMode(LED_PIN, OUTPUT);
  Serial.begin(9600);
  Serial.println("Knock Detector Ready");
}

void loop() {
  int sensorVal = analogRead(PIEZO_PIN);
  
  // Check if vibration strength exceeds our threshold limit
  if (sensorVal > KNOCK_THRESHOLD) {
    digitalWrite(LED_PIN, HIGH);
    Serial.print("Knock Detected! Strength: ");
    Serial.println(sensorVal);
    
    delay(100); // Keep LED on briefly so it is visible
    digitalWrite(LED_PIN, LOW);
  }
}
```

## Hardware & Safety Concept: Piezoelectricity
A **piezoelectric sensor** works by converting mechanical stress (like a tap, bend, or vibration) into electrical energy. When you tap the ceramic crystal inside the piezo element, it deforms slightly and generates a small electrical charge. Because these spikes can exceed 5V and potentially damage the microcontroller input pins, a high-value parallel resistor (e.g. 1 Megaohm) is used on real hardware to bleed off excess charge.

## Try This! (Challenges)
1. **Calibrate Sensitivity**: Test different tap strengths. Adjust `KNOCK_THRESHOLD` in the code to make the sensor more or less sensitive.
2. **Knock Alarm**: Connect a Buzzer to D8 and play a brief high-pitched tone (`tone(8, 2000, 100);`) whenever a knock is detected.

## Troubleshooting
| Symptom | Likely Cause | Fix |
| --- | --- | --- |
| LED flashes constantly | Threshold is set too low | Raise `KNOCK_THRESHOLD` to a higher number (e.g. 200 or 300) to filter out background vibration noise. |
| LED never flashes | Piezo is connected backwards or wrong threshold | Make sure POS is connected to A0 and NEG is connected to GND. Check that your tap slider value is high enough. |
