# 38 - Pico Piezo Vibration

Read analog shock values from a piezo vibration sensor module.

## Goal
Learn how to read transient analog voltage spikes generated by piezoelectric transducers using the Pico's ADC.

## What You Will Build
A shock impact meter:
- **Piezo Vibration Sensor (GP26)**: Connected to ADC0. Tapping or shaking the sensor outputs varying analog spikes printed to the Serial Monitor.

| Component | Component Pin | Pico Pin | Notes |
| --- | --- | --- | --- |
| Piezo Sensor | VCC | 3.3V | Power supply |
| Piezo Sensor | OUT (Analog) | GP26 | Analog shock output |
| Piezo Sensor | GND | GND | Ground return |

## Code
```cpp
const int PIEZO_PIN = 26; // GP26 (ADC0)

void setup() {
  Serial.begin(9600);
  pinMode(PIEZO_PIN, INPUT);
  Serial.println("Piezo Shock Monitor Online");
}

void loop() {
  int shockVal = analogRead(PIEZO_PIN);

  // Print values above noise floor
  if (shockVal > 100) {
    Serial.print("Shock Intensity: ");
    Serial.println(shockVal);
  }

  delay(100); // 10 samples per second
}
```

## Hardware & Safety Concept: Piezoelectric Effect
Piezoelectric materials (like quartz crystals or specialized ceramics) generate an electrical charge when mechanically deformed (compressed, bent, or vibrated). Because these materials can produce high voltage spikes when hit hard, commercial sensor modules include internal zener diodes and resistor shunts to clamp the voltage to 3.3V, protecting the sensitive microcontroller pins.
