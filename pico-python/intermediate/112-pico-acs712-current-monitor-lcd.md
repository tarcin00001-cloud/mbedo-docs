# 112 - Pico ACS712 Current Monitor LCD

Build a real-time current monitoring console that reads an ACS712 hall-effect current sensor, calculates amperage, and displays live readings on an I2C character LCD screen.

## Goal
Learn how to read an analog voltage from an ACS712 current sensor, apply the sensitivity conversion formula to calculate current in amps, and display formatted current readings on an I2C character LCD in MicroPython.

## What You Will Build
A current measurement console:
- **ACS712 Current Sensor (GP26)**: Measures current flowing through the sensing circuit.
- **I2C 16x2 LCD (GP4, GP5)**: Displays the measured current in amps and a load status indicator.

## Parts Needed
| Part | MbedO Component Type | Required in MbedO | Required on hardware |
| --- | --- | --- | --- |
| Raspberry Pi Pico | `pico` | Yes | Yes |
| ACS712 Current Sensor (5A or 20A module) | `potentiometer` | Yes (represented by potentiometer slider) | Yes |
| I2C 16x2 LCD Display | `lcd` | Yes | Yes |

## Wiring

| Component | Component Pin | Pico Pin | Wire Colour | Notes |
| --- | --- | --- | --- | --- |
| ACS712 Module | VCC (+) | 5V (VBUS) | Red | Sensor power (5V required) |
| ACS712 Module | GND (−) | GND | Black | Ground reference |
| ACS712 Module | OUT (Analog) | GP26 | Yellow | Analog voltage output |
| I2C LCD | SDA / SCL | GP4 / GP5 | Shared Orange/Blue | Shared I2C screen bus |
| All Modules | GND | GND | Black | Shared return path to ground |

> **Wiring tip:** Connect the ACS712 module VCC to 5V VBUS, and the analog output to GP26. Connect the LCD to GP4/GP5. All grounds are shared. The load current wires pass through the ACS712 brass block terminals (not through the Pico).

## Code
```python
from machine import Pin, ADC, I2C
import utime
from machine_lcd import I2cLcd

acs712 = ADC(26) # GP26 = ADC channel 0

# Initialize I2C Bus 0 on GP4/GP5
i2c = I2C(0, sda=Pin(4), scl=Pin(5), freq=400000)
lcd = I2cLcd(i2c, 0x27, 2, 16)

# ACS712-5A sensitivity: 185 mV/A. For 20A module use 100 mV/A.
SENSITIVITY_MV_A = 185.0

# Zero-current reference voltage (Vcc/2 = 2.5V, or ~32767 on 16-bit ADC at 3.3V ref)
# NOTE: the ACS712 outputs 0V to Vcc; zero current = Vcc/2 = 2.5V
# With 5V sensor on 3.3V ADC reference, the mid-point maps to ~50000 raw counts
ZERO_CURRENT_RAW = 32767 # Calibrate this for your hardware setup

# Oversampling: average 50 readings to reduce noise
SAMPLES = 50

def read_current():
    total = 0
    for _ in range(SAMPLES):
        total += acs712.read_u16()
    avg_raw = total / SAMPLES
    
    # Convert raw ADC value to voltage (0 - 3.3V)
    voltage_v = avg_raw * 3.3 / 65535
    
    # Zero-current voltage is approximately 2.5V (half of 5V supply)
    zero_v = ZERO_CURRENT_RAW * 3.3 / 65535
    
    # Calculate current in Amps
    delta_v_mv = (voltage_v - zero_v) * 1000.0
    current_a = delta_v_mv / SENSITIVITY_MV_A
    return current_a

lcd.clear()
lcd.move_to(0, 0)
lcd.putstr("Current Monitor")
lcd.move_to(0, 1)
lcd.putstr("Initializing...")
utime.sleep(1)

print("ACS712 Current Monitor active.")

while True:
    current_a = read_current()
    abs_current = abs(current_a)
    
    # Determine load status
    if abs_current < 0.1:
        status = "NO LOAD"
    elif abs_current < 2.0:
        status = "LIGHT"
    elif abs_current < 4.0:
        status = "MEDIUM"
    else:
        status = "HIGH"
        
    # Update LCD Display
    lcd.clear()
    lcd.move_to(0, 0)
    lcd.putstr("I: {:.3f} A".format(abs_current))
    lcd.move_to(0, 1)
    lcd.putstr("Load: " + status)
    
    print("Current: {:.3f} A | Load: {}".format(abs_current, status))
    utime.sleep_ms(500)
```

## Code Walkthrough
| Expression | What It Does |
| --- | --- |
| `total / SAMPLES` | Averages 50 ADC readings to smooth out electrical noise on the current sensor output. |
| `delta_v_mv / SENSITIVITY_MV_A` | Applies the ACS712 sensitivity coefficient (185 mV/A for the 5A module) to convert voltage change to current. |

## Hardware & Safety Concept: Hall Effect Current Sensing
The ACS712 uses a Hall effect element to measure magnetic fields generated by current flow. Because the measurement is **galvanically isolated** (the sensing element never contacts the power circuit), it is safe to measure mains or high-voltage current without exposing the microcontroller to dangerous voltages. Always ensure the load current remains within the sensor's rated range.

## Troubleshooting
| Symptom | Likely Cause | Fix |
| --- | --- | --- |
| Current reading always 0 | Zero-current calibration mismatch | Adjust `ZERO_CURRENT_RAW` to match the actual ADC reading when no current flows. |
| Reading fluctuates rapidly | Insufficient averaging | Increase `SAMPLES` from 50 to 100 for better noise reduction. |
