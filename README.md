# MbedO Project Library

125 step-by-step Arduino Uno projects for the MbedO interpreted runtime.

All projects run in **Interpreted AVR subset** mode — no backend compile required.
Code in every project is written to the interpreted mode rules: no `for` loops, no arrays, no custom helper functions.
